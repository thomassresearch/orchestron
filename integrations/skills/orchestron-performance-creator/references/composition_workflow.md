# Composition Workflow

Use this reference when creating a new Orchestron performance or reshaping a draft into an electronic-music form. The goal is to turn musical intent into pattern pads, pad loops, controller curves, and score specs without loading genre detail that is not needed.

## Choose the formal promise first

Before writing notes, decide how the piece should govern expectation:

- Ambient: soften expectation through slowly changing states, sparse gestures, drones, and open-ended duration.
- Techno: lock the listener into a stable groove, then create form through small process changes, accretion, subtraction, and altered emphasis.
- Trance: dramatize expectation through breakdowns, buildups, and clear arrivals where the full groove returns.
- Goa/psytrance: create long rises through relentless pulse, scalar riffing, accumulative motion, simplification, and renewed climbs.

This choice controls how much contrast to write, how quickly phrases should turn over, and whether the listener should feel suspended, locked in, or repeatedly released.

## Build one core loop

Write one loop that already contains the genre contract:

- Techno and psy: the 1-2 bar groove must already work before arrangement begins.
- Trance: the core loop should imply the later arrival, because the drop only works if it returns to something credible.
- Ambient: the core object can be a drone-plus-gesture state instead of a beat loop.
- Goa: the core object is often a riff ecology: bass motor, modal/scalar lead cell, counter-layer, and subdivision texture.

Avoid solving a weak core loop by adding unrelated layers. A layer should change pulse, subdivision, anticipation, color, or release.

## Define harmonic state

Use the smallest number of harmonic states that can differentiate the sections. If Orchestron triggers chord qualities directly, treat supported labels such as `min`, `maj`, `min7`, `maj7`, `dom7`, `m7b5`, `sus2`, and `sus4` as color states rather than as a progression-writing obligation. If a target surface later supports extra colors such as add9, use them sparingly as color states too.

Prefer smooth voice-led changes, common tones, and useful inversions over maximal chord density. Denser harmony usually works best in ambient states, intros, or breakdowns; exposed grooves often work better with simpler bass functions and lighter upper color.

## Create variants, not unrelated ideas

For each important pattern family, make a small set of related variants:

- Base: the identity loop.
- Reduced: removes a structural layer such as kick, bass, lead, or harmonic bed.
- Heightened: adds subdivision, register lift, extra chord color, or an active arp.
- Tension: uses denser repetition, rising controller motion, snare or hat activity, suspended bass, or a delayed resolution.

This is usually enough to create 16- and 32-bar form while keeping identity intact. Dance arrangement often comes from which layers are present, not from constantly introducing new thematic material.

## Map Orchestron hierarchy to phrase scale

Use the pattern system as a musical hierarchy:

- Cell: 1-2 bar identity loop.
- Pattern: 4-bar statement or stable pad behavior.
- Phrase: 8 bars.
- Section: 16 or 32 bars.
- Supergroup: macro-function such as intro, core, breakdown, buildup, drop, plateau, or outro.

Build pad-loop groups around functions rather than instruments. Good groups include metric anchor, subdivision layer, syncopated counterforce, harmonic bed, lead motive, transition signal, reduced texture, and tension form.

## Arrange by subtraction and return

Once the core loop and variants exist, duplicate the main material across the form and use controlled addition/subtraction:

- Add layers in 8- or 16-bar spans.
- Remove kick or bass before an arrival if the genre needs release.
- Reintroduce familiar material as the release, not as a brand-new idea.
- Use controller curves for filter, density, brightness, and register motion.
- Reserve new motives for major sectional changes or late climaxes.

Judge the result by tension management rather than idea count. The strongest pattern-based performances know when to keep the beat explicit, when to withdraw it, when to let harmony stay still, when to intensify subdivision, and when to let a familiar loop return.
